# MaycleMay Messenger 💬

Реальный десктопный мессенджер для общения с друзьями через интернет!

## ✨ Возможности

- 📝 Регистрация с ником, юзернеймом и фото профиля
- 🔍 Поиск друзей по юзернейму
- 💬 Обмен сообщениями в реальном времени
- 😊 Отправка смайликов (200+ эмодзи)
- 🌐 Работает через интернет - общайтесь с друзьями из любой точки мира
- 🟢 Статусы онлайн/оффлайн
- ⌨️ Индикатор "печатает..."
- 📱 Нативное десктопное приложение

## 🚀 Быстрый старт

### Для разработчиков

1. **Установите зависимости:**
```bash
npm install
```

2. **Разверните сервер в облаке** (см. DEPLOY.md)
   - Рекомендуется: Render.com (бесплатно)
   - Альтернативы: Railway.app, Glitch.com

3. **Настройте URL сервера в app.js:**
```javascript
const SERVER_URL = 'https://ваш-сервер.onrender.com';
```

4. **Соберите приложение:**
```bash
npm run build-win    # для Windows
npm run build-mac    # для macOS
npm run build-linux  # для Linux
```

5. **Отправьте друзьям** установочный файл из папки `dist/`

### Для пользователей

1. Скачайте установочный файл от друга
2. Установите приложение
3. Запустите MaycleMay
4. Введите свой ник и @юзернейм
5. Найдите друзей по их @юзернейму
6. Начните общаться!

## 📖 Подробная инструкция

Смотрите файл **DEPLOY.md** для детальной инструкции по развертыванию сервера.

## 🛠 Разработка

Для локального тестирования:

1. Запустите сервер:
```bash
npm run cloud-server
```

2. В app.js установите:
```javascript
const SERVER_URL = 'http://localhost:3000';
```

3. Запустите приложение:
```bash
npm start
```

## 💡 Как это работает

1. **Облачный сервер** - размещается на бесплатном хостинге (Render, Railway)
2. **Десктопное приложение** - подключается к серверу через WebSocket
3. **Все друзья** подключаются к одному серверу
4. **Сообщения** передаются в реальном времени через интернет

## 🔧 Технологии

- **Electron** - нативное десктопное приложение
- **Socket.io** - WebSocket для реального времени
- **Node.js + Express** - облачный сервер
- **HTML/CSS/JavaScript** - интерфейс

## ⚠️ Важно

- Все пользователи должны использовать приложение с одинаковым URL сервера
- Бесплатные серверы могут "засыпать" - первое подключение займет 30 секунд
- Для постоянной работы рекомендуется платный хостинг ($5/месяц)
